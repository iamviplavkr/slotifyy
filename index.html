Slotify aims to develop a secure, real-time smart parking management system that allows users to book parking slots digitally, perform authenticated payments using JWT security, and enables administrators to manage live parking data efficiently.

Problem Statement

Urban areas face major parking management issues due to lack of real-time availability systems, manual booking processes, and insecure payment mechanisms. Users struggle to find available slots, leading to traffic congestion and time wastage. Slotify addresses this by providing a real-time slot booking and secure payment system integrated with backend database and JWT authentication.

Literature Review / Market Research

Existing systems mainly provide static parking information without real-time updates or secure authentication. Many applications lack backend integration with proper payment validation and role-based admin control.

Research Gap / Innovation

Slotify integrates Android frontend (Kotlin), .NET 8 Web API backend, SQL Server database, Firebase OTP login, and JWT-based authorization. It ensures secure payment APIs and real-time admin data management.

System Architecture

Database Layer

SQL Server database storing Users, Parking Slots, Bookings, and Payments with Stored Procedures for secure transactions.

Backend API

.NET 8 Web API with JWT Authentication, Firebase OTP verification, and secure Booking & Payment endpoints.

Android Application

Kotlin-based Android app enabling real-time slot booking, secure payments, and admin dashboard integration.
